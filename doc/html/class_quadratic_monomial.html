Detailed Description

Quadratic monomials.

Evaluates quadratic monomials in dim dimensions. Monimial 0 = 1 Monomial 1 ... dim = x_i Monomial dim+1 ... 2*dim = 0.5*x_i^2 Monomial 2*dim+1 ... 3*dim = x_1*x_2 ... x_1*x_n Monomial 3*dim+1 ... 4*dim-1 = x_2*x_3 ... x_2*x_n ... Monoial (dim^2 + 3*dim +2)/2 = x_{n-1}x_n
